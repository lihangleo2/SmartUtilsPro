
# <center>smart-utils (API12 - Dev: 5.0.3.600)</center>

一个智能的常用工具类库，让你少走鸿蒙开发弯路。

+ 极简api
+ 几行代码即可实现功能
+ 持续更新

## 安装

```typescript
ohpm i @leo/smart_utils
```

## API介绍

以下配置中如果有和[全局配置](#全局配置)一样的名字函数，此次弹窗属性会按照本次配置生效

| 模块                       | 介绍     |
|:-------------------------|:-------|
| SmartPermissionUtil         | 权限请求框架 |
| SmartTimer              | 倒计时工具类 |
